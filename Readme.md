## Задание 2 (практическое)

Необходимо написать запросы для MongoDB, работающие с коллекцией `books`.

### Структура документа коллекции books

```json
{
  title: "string",
  description: "string",
  authors: "string"
}
```

### 1. Запрос(ы) для вставки данных минимум о двух книгах

```javascript
// Вставка нескольких книг 
db.books.insertMany([
  {
    title: "Преступление и наказание",
    description: "Социально-психологический роман об убийстве и последующих моральных страданиях студента Раскольникова",
    authors: "Фёдор Достоевский"
  },
  {
    title: "Мастер и Маргарита",
    description: "Мистический роман, сочетающий сатиру на советское общество с библейским сюжетом и историей любви",
    authors: "Михаил Булгаков"
  }
]);
```

### 2. Запрос для поиска полей документов коллекции books по полю title

```javascript
// Поиск точного совпадения по названию
db.books.find({ title: "Преступление и наказание" });

// Поиск с использованием регулярного выражения (частичное совпадение, регистронезависимо)
db.books.find({ title: { $regex: "Мастер", $options: "i" } });

// Поиск с проекцией (возврат только нужных полей, без _id)
db.books.find(
  { title: "Мастер и Маргарита" },
  { title: 1, description: 1, authors: 1, _id: 0 }
);
```

### 3. Запрос для редактирования полей: description и authors коллекции books по _id записи

```javascript
// Обновление конкретной книги по _id (замените на реальный ObjectId)
db.books.updateOne(
  { _id: ObjectId("67f5a1b2c3d4e5f6a7b8c9d0") },
  {
    $set: {
      description: "Обновлённое описание книги (новое издание)",
      authors: "Фёдор Достоевский (обновлённая редакция)"
    }
  }
);
```
